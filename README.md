# Proposal Pro

A professional proposal generator with dark/light mode and PDF export functionality.

## Features

- 📝 Rich text editing with professional formatting
- 🎨 Dark/Light mode toggle
- 📄 PDF and HTML export
- 📱 Responsive design
- 🚀 Fast and lightweight
